# ecoSUB-toolbox

Code for working with ecoSUB micro.

Reads data from C3 GUI log files C:\ecoSUB C3 GUI\Logs

## MATLAB

Wrapper scipt is  ecoSUB_process.mlx which calls

 * ecoSUB_scatter.mlx
 * ecoSUB_profile.mlx
 * ecoSUB_plot_engineering.m
 * ecoSUB_track.mlx
